# suanpan-tools

## 简介

这是一个用于算盘客户端的更新工具，可以查看，导入，导出算盘的相关组件和模板，基于python开发。

## 开始使用

- 运行`pip install -r requirements.txt`即可安装所有依赖
- 使用`pip install --editable .`进行打包安装
- 即可在文件下的命令行中使用算盘工具相关的指令
